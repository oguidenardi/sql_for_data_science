## Data Cleaning and Export

Nesta etapa aplico técnicas de **limpeza de dados**
para preparar informações para exportação e consumo
em ferramentas externas.

### Conceitos praticados
- REGEXP_REPLACE
- Tratamento de strings
- Preparação de dados para CSV
- Qualidade de dados

Esse tipo de consulta é comum em pipelines de dados
e processos de ETL.
