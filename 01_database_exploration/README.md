## Database Exploration

Nesta etapa realizo a exploração inicial do banco de dados,
identificando tabelas disponíveis e entendendo a estrutura
da tabela `employees`.

### Conceitos praticados
- Exploração de schema
- Estrutura de tabelas
- Tipos de dados

Essa etapa é fundamental para qualquer projeto de Análise de Dados,
pois permite compreender o contexto antes de iniciar análises.
