## Basic Queries

Esta etapa aborda as consultas básicas em SQL,
essenciais para a extração inicial de dados.

### Conceitos praticados
- SELECT
- Seleção de colunas
- ORDER BY
- LIMIT

### Habilidades desenvolvidas
- Leitura de tabelas
- Controle de volume de dados
- Ordenação para identificação de padrões

As consultas utilizam dados da tabela `employees`,
simulando cenários comuns de exploração inicial.
