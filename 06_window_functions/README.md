## Window Functions

Nesta etapa utilizo **Window Functions** para análises avançadas,
permitindo cálculos analíticos sem o uso de subqueries complexas.

### Conceitos praticados
- LAG
- ROW_NUMBER
- DENSE_RANK
- OVER()
- PARTITION BY

### Aplicações
- Ranking de clientes
- Análise temporal
- Comparação entre eventos sequenciais

Essas técnicas são amplamente utilizadas em projetos
de Análise de Dados e Ciência de Dados.
